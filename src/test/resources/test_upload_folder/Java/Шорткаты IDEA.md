Alt + 1: Открыть список файлов
Alt + insert: вызвать менюшку (в ней можно искать)
Shift Shift: поиск везде
Ctrl + Alt + V: объявить переменную
В конце можно добавить .var и это тоже создаст переменную
Ctrl + Space: предложения, если класс не заимпорчен, то два раза и покажет все
Ctrl + Shift + Space: выдаст конкретные подходящие результаты
Ctrl + Alt + O: Optimize imports
Ctrl + Alt + N: inline variable

Ctrl + Q: Fast documentation
Ctrl + B: Вместо Ctrl + click (переход в класс или метод)
Ctrl + Alt + ->\<-: навигация по истории (вперед назад)
Ctrl + E: история посещений 
Ctrl + Shift + E: история посещений с позицией и кодом
Ctrl + F11: сделать закладку
Shift +F11: посмотреть закладки

можно писать не только тудушки, а еще и кастомить свои 
можно создавать scratches для удобства (типа спрятанные файлы, мусорка всякого)

Ctrl + K: сделать коммит
